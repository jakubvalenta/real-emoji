Installation

{{ site.title }} are available as an emoji font, which is also how the standard emojis work. The font is based on Twitterâ€™s emoji set, therefore it contains not only the new emojis but also all the standard ones.

Windows

  1. Download Emoji.ttf.
  2. Copy the font file to the Fonts folder.
  3. See Twemoji Color Font - Install on Windows for more information.

macOS

  1. Download Emoji-mac.zip.
  2. Install the font file using FontBook and accept the warning.
  3. See Twemoji Color Font - Install on OS X for more information.

Linux

  1. Download Emoji.ttf.
  2. Copy the file to ~/.fonts/.
  3. See Twemoji Color Font - Install on Linux for more information.

iOS

  1. Your device needs to be jailbroken. This poses a security risk, so unless your device is already jailbroken, we don't suggest jailbreaking it just to install some emojis.
  2. Install the app BytaFont from a Cydia repository.
  3. Use the app BytaFont to replace the default emoji font.
  4. Please search the internet for more information.

Android

  1. Your device needs to rooted. This poses a security risk, so unless your device is already rooted, we don't suggest rooting it just to install some emojis.
  2. Download Emoji.ttf.
  3. Copy the font file using adb to /system/fonts and replace the default emoji font.
  4. See How to Install Custom Fonts on your Android Device for more information.

Making {{ site.title }} official

How could we get these new emojis to all phones and computers without the need for the complicated installation? To answer this question we need to first introduce a few technical details:

How do emojis work

Emojis are characters like letters, numbers or punctuation. There is one list called Unicode which contains all characters of all scripts (latin, chinese...) including emojis. All software manufacturers use the Unicode list to ensure compatibility between their programs. This way text writen in one computer program displays the same in a different program. Which also means that an emoji created on an iPhone looks similarly when displayed on an Android phone and vice versa.

How are new emojis added

The Unicode list of characters is regularly updated to add new characters (e.g. of historic scripts) and also new emojis. This is a bureaucratic process managed by the Unicode Consortium, which is an organization in which all major companies (Apple, Google, Facebook, Microsoft etc.) have their seat.

Any person can submit a proposal for a new emoji to the Unicode Consortium and it will be assesed using a defined process.

Which emojis can be accepted?

There are many selection factors. For example:

  • High expected frequence of use
  • Not already representable by existing emoji
  • Doesn't show specific person, logo, building etc.
  • Manufactures are expected to implement the emoji
  • and others

Submitting {{ site.title }}

We can see now that to make the new emojis official, they would need to meet the selection criteria of the Unicode Consortium and be submitted using the defined process.

I intend to submit only those emojis that have higher chance of being accepted. I do not want to bother the Unicode Consortium just to show how the submission process works or to make some point.
