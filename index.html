Download

Emojis for grown-ups are currently available as an Emoji font. This font is based on Twitter’s emojis so it also contains all standard emojis.

  • Linux and Android: Emoji.ttf
  • Webfont: Emoji.zip
  • Windows: Not technically possible, because Windows doesn’t support the format of Twitter’s emoji font.
  • iOS: Not technically possible, because Apple doesn’t allow changing the system font.
